# diyyick
A very simple PHP MVC framework
PHP MVC library for small to medium PHP application
Install using composer
folder structure 
project/app
           /controllers
           /models 
           /views
       /config 
              /authconfig.php
              /genconfig.php
       /public
              /index.php
              /.htaccess
       /vendor/
       /.htaccess
May include router file in the controllers folder
